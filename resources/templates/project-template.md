## Instructions for Use

1. **Fill in all sections** with specific, concrete details
2. **Focus on measurable outcomes** whenever possible
3. **Be honest and accurate** - only include what you can discuss confidently
4. **Keep technical depth appropriate** for the intended audience
5. **Review for consistency** with other project documentation

## Integration with CV System

Once completed:
1. Add project to `resources/experience/projects.yaml`
2. Update relevant roles in `resources/experience/roles.yaml`
3. Create evidence file in `resources/evidence/`
4. Update relevant CV files to include the project
5. Log the addition in `resources/adapted-profiles-log.yaml`
